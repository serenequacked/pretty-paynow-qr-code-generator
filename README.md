# pretty-paynow-qr-code-generator

Generates a pretty QR codes for use. 

## Attribution

This project incorporates and builds upon code from
[xkjyeah/paynow-qr-generator](https://github.com/xkjyeah/paynow-qr-generator).

The original work provides the core PayNow QR generation logic.
This repository extends it with a template system, customisable
backgrounds, and print-ready layouts focused on user experience
and visual design.

## Project Philosophy

PayNow QR codes are often functional but visually dull.
This project treats QR codes as **design objects**, not just payment tools.

The goal is to make it easy for anyone to generate QR codes that feel:
- Friendly
- Fun
- On-brand
- Suitable for gifts, stalls, and events
